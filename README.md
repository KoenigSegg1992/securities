## 一般性货币政策工具

### 存款准备金政策（最猛烈）

法定存款准备金率越高，商业银行上交的钱越多，商业银行可用资金变少，贷款或投资变少，市场供应量变少

### 再贴现政策（主要）

再贴现率提高，商业银行再贴现钱变少，融资成本增加，可用资金变少。

### 公开市场业务（常用）

央行买入证券，增加货币供应量

央行卖出证券，减少货币供应量
